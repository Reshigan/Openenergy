Ten ground-up frontend designs — one tailored to each role's job, environment, and device habits. Not constrained by the current Meridian chrome. The spine of every design is the role's real user journeys, surfaced as packaged, metered products — because module complexity is what we charge for.

How journeys are priced

Each role's hero journeys carry a tier badge mapped to the platform's feature-depth rubric. The tier sets the price; the metered unit sets the meter. The badges and metered units appear inside every mockup so the value a customer pays for is legible in the product itself.

Foundation · L3

State machine + audit

Validated transitions, server-side rules, an audit trail. The entry tier — a journey that's real, not a form.

Professional · L4

Full workflow

Pre-trade gating, downstream cascades, calendar/timer-driven SLAs, structured reason codes, dunning & escalation, evidence chain.

Regulator-grade · L5

Certified & reconciled

Tamper-evident audit, certified exports (NERSA / EMIR / Verra), reconciliation against external systems. Top tier.
