Eleven ground-up frontends — one per role's job, environment, and device habits. This v2 set carries one obsession through every screen: ease. Each design names the single hardest thing it makes effortless and builds the whole surface around it. Still 2–3 distinct directions per role, each as web + mobile PWA, spined on the role's real journeys surfaced as metered, priced products.

How journeys are priced

Each role's hero journeys carry a tier badge mapped to the platform's feature-depth rubric. The tier sets the price; the metered unit sets the meter. Badges and metered units appear inside every mockup so the value a customer pays for is legible in the product itself.

Foundation · L3

State machine + audit

Validated transitions, server-side rules, an audit trail. The entry tier — a journey that's real, not a form.

Professional · L4

Full workflow

Pre-trade gating, downstream cascades, calendar/timer-driven SLAs, structured reason codes, dunning & escalation, evidence chain.

Regulator-grade · L5

Certified & reconciled

Tamper-evident audit, certified exports (NERSA / EMIR / Verra), reconciliation against external systems. Top tier.
