System Operator Control Surface

NTCSA / national system operator. Real-time, high-consequence dispatch and constraint management — every screen is a decision under a clock.

Persona

System operator at NTCSA running the national transmission network from a 24/7 control room. Lives in dispatch nominations, reserve activation, load curtailment under load-shedding stages, grid-code conformance and the connection/capacity queue. Big wallboard screens at the desk; field acknowledgements on mobile when away from the console. Stress is constant — a missed curtailment SLA is a stability and statutory event, so the UI must make the active, time-critical item impossible to miss and one action away.

Why these 3 directions

Control rooms run on glanceability and urgency. Wallboard = always-on status density for the big screen. Grid map = spatial truth, because constraints and curtailment are geographic. Incident command = the active event becomes a timed, escalation-aware command surface. Mobile in every option is a one-thumb field acknowledge / curtailment confirm.
